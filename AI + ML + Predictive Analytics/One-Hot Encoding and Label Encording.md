Label Encoding simply encodes each category (categorical data) as an integer value. Sklearn provides a preprocessing library to assist with this. 

~~~
import matplotlib.pyplot as plt
import numpy as np
import pandas as pd

brain = pd.read_csv('Resources/brain_categorical.csv')
brain.head()

X = brain[["gender", "age", "size"]]
y = brain["weight"].values.reshape(-1, 1)
print(X.shape, y.shape)

from sklearn.preprocessing import LabelEncoder
label_encoder = LabelEncoder()

data = X.copy()

label_encoder.fit(data["gender"])
label_encoder.classes_
